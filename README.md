# httk-web

The website for httk (served at http://httk.org).

## Layout

The site is generated by `publish_static.py` (using the httk₂ `httk-web`
static site generator, imported as `httk.web`) from two independent source
trees into `docs/` (git-ignored output):

- `src/` &rarr; `docs/` — the **httk₂ main site** (the current main version).
- `src-v1/` &rarr; `docs/v1/` — the **legacy httk version 1** subsite, served at
  `/v1/`. It keeps v1-accurate content, its own reduced-nav templates with a
  legacy banner, and links docs to `https://olddocs.httk.org`.

Each tree has its own `config.httkweb`, `content/`, `templates/`, and `static/`.

## Building

The `httk-web` generator is declared as a dependency in `pyproject.toml`.
Install it (preferably into a virtual environment) and build both trees:

```bash
pip install .
python3 ./publish_static.py
```

Then open `docs/index.html` in your browser.

## Developing

A `Makefile` provides the common targets:

```bash
make generate      # build docs/ (and docs/v1/) with publish_static.py
make serve         # run the dynamic dev server (serve_dynamic.py) on :8080
make serve_static  # build, then serve docs/ over http.server on :8080
make clean
```
