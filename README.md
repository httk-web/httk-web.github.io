# httk-web

The website for httk (served at http://httk.org).

## Layout

The site is generated by `publish.py` (using the httk₁ `httk.httkweb` static
generator) from two independent source trees into `docs/` (git-ignored output):

- `src/` &rarr; `docs/` — the **httk₂ main site** (the current main version).
- `src-v1/` &rarr; `docs/v1/` — the **legacy httk version 1** subsite, served at
  `/v1/`. It keeps v1-accurate content, its own reduced-nav templates with a
  legacy banner, and links docs to `https://olddocs.httk.org`.

Each tree has its own `config.httkweb`, `content/`, `templates/`, and `static/`.
Build both with:

```bash
PYTHONPATH=/path/to/httk/src python3 ./publish.py
```
